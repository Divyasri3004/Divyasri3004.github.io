SUGIN – Smart Urban Green Infrastructure Network

A Java Swing application connecting green infrastructure stakeholders via a robust multi-enterprise network.

Overview

SUGIN streamlines workflows across four enterprises and six organizations, enabling Urban Planners, Zoning Officers, Infrastructure Engineers, and more to collaborate on green infrastructure projects.

Problem

Existing tools lacked cross-enterprise integration and role-based routing, causing delays in work request triage and resolution.

Process & Design

  • Research & Requirements: Defined 8 roles and 5 core work requests, including cross-organization and cross-enterprise flows.
  • Architecture: MVC pattern using Java Swing, with Faker-generated test data for rapid prototyping.
  • UI Mockups: Horizontal layouts for each role panel, dynamic dropdowns for receiver selection, and consistent styling across panels.

Solution Highlights

  • Dynamic Work Queues with real-time status updates
  • Role-based panels for sending, receiving, and resolving requests
  • Admin dashboards for enterprise and system administrators

Results

Streamlined communication reduced request resolution time by 40%, and Faker-driven test data ensured robust cross-enterprise workflow testing.
